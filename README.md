# INM420 Three.js Interactive Demo

This project is a learning exercise for INM420 to practice rendering and interacting with 3D content in the browser using Three.js.

## Learning Goals

- Set up a Three.js project with a modern JavaScript workflow (Vite).
- Build a basic scene with a camera, lighting, and a 3D object.
- Animate the object with a continuous render loop.
- Add user interaction with camera controls and click picking.

## Features

- Interactive 3D object rendered with Three.js.
- Orbit camera controls (drag to rotate, scroll to zoom).
- Click interaction using raycasting (color toggle + pulse scale effect).
- Responsive canvas that resizes with the browser window.

## Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start development server:

   ```bash
   npm run dev
   ```

3. Build production bundle:

   ```bash
   npm run build
   ```

4. Preview production build:

   ```bash
   npm run preview
   ```
